# Image Scaler – Масштабирование изображений БЕЗ "МЫЛА"

## Описание

**Image Scaler** — это десктопное приложение с графическим интерфейсом для масштабирования изображений.
Главная особенность: используется алгоритм **ближайшего соседа (NEAREST)**, который **не размывает** изображение при масштабировании и сохраняет чёткость пикселей.
Это идеальный выбор для:

- Пиксель-арта
- Иконок и спрайтов
- Схем, чертежей, сканов документов
- Любых изображений, где важна резкость границ

Приложение поддерживает перетаскивание файлов (drag‑and‑drop), точную настройку размеров и предпросмотр результата с зумом.

## Возможности

- Drag‑and‑drop – перетащите файл прямо в окно
- Загрузка через стандартный диалог выбора файла
- Задание масштаба коэффициентом (например, ×2, ×0.5)
- Задание точных размеров (ширина и высота)
- Автоматическое переключение режимов «Увеличение» / «Уменьшение»
- Предупреждение при нецелом делении (для уменьшения)
- Предпросмотр результата с возможностью зумирования (от 10% до 1600%)
- Сохранение: перезапись исходного файла или сохранение в новый файл
- Поддержка популярных форматов: PNG, JPEG, BMP, GIF, TIFF

## Как это работает без замыливания?

В библиотеке Pillow при масштабировании используется метод ближайшего соседа (NEAREST). Он не усредняет цвета соседних пикселей, а просто копирует их. В результате при увеличении каждый пиксель оригинала превращается в блок пикселей того же цвета, что даёт резкий, «пиксельный» вид без сглаживания и размытия.

```python
# Ключевая строка в коде:
scaled_image = original_image.resize((new_width, new_height), Image.NEAREST)
```

## Требования для запуска .py-скрипта

- Python 3.7 или новее
- Установленные зависимости (перечислены в requirements.txt)

## Установка и запуск

- Клонируйте репозиторий
- (Если запускать из .py-файла) Установите зависимости из requirements.txt и запустите .py-скрипт.
- (Если запускать из .exe-файла) Запустите .exe-программу.

## Инструкция по использованию

1. Загрузите изображение – перетащите файл в левую область или нажмите на неё и выберите файл в проводнике.
2. Настройте масштабирование – введите коэффициент (например, 2 для увеличения вдвое) – режим определится автоматически. Или вручную укажите желаемые ширину и высоту – коэффициент пересчитается сам. При уменьшении до нецелых размеров появится предупреждение.
3. Запустите масштабирование – нажмите кнопку «МАСШТАБИРОВАТЬ!». Автоматически откроется вкладка «Превью».
4. Оцените результат - если всё устраивает, нажмите «Сохранить».
5. Сохраните результат – выберите «Перезаписать оригинал» или «Сохранить как новый файл».

## Важные замечания

- При уменьшении изображения метод ближайшего соседа может давать «лестничный» эффект (алиасинг). Это особенность метода, исключающая сглаживание.
- Для фотографий с плавными градиентами лучше использовать другой алгоритм (например, LANCZOS), но данное приложение намеренно выбирает NEAREST для сохранения резкости.
- Поддерживаются все форматы, которые понимает Pillow.
